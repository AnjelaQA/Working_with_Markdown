# подсказки по командной строке

команда смены дируктории
```sh
cd c:\folder_name
```

команда отображения текущей дируктории
```sh
pwd
```

листинг текущей директории

Windows:
```sh
dir
```
MacOs, Linux:
```sh
ls
```

Удаление файла

Windows:
```sh
del <filename>
```
MacOs, Linux:
```sh
rm <filename>
```
